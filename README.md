# 🌿 EcoChampions: Gamified Environmental Education

**Theme:** Smart Education  
**Project:** Gamified Environmental Education Platform for Schools and Colleges

EcoChampions is an interactive web platform designed to transform traditional environmental education into an engaging, competitive, and action-oriented experience. By combining dynamic quizzes, real-world habit tracking, and competitive leaderboards, this platform builds genuine passion for sustainability.

---

## ✨ Key Features

- **🎮 Gamified Learning (Missions):** Interactive quizzes loaded dynamically from a dataset. Users earn "Leaf Points" for correct answers and unlock "Fun Facts" to ensure immediate knowledge retention.
- **🌍 Real-World Action Tracking:** A dashboard feature that bridges the gap between digital learning and physical action by rewarding users for logging daily eco-friendly habits (e.g., using a reusable water bottle).
- **🏆 Global Leaderboard:** Drives engagement through healthy competition, allowing students to see how their impact stacks up against peers.
- **🔐 Secure User Profiles:** Full authentication system using SQLite and password hashing to save individual user progress and scores across sessions.

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Database:** SQLite (built-in, auto-generating)
- **Frontend:** Vanilla HTML5, CSS3, JavaScript
- **Data Handling:** CSV-based dynamic content loading

## 🚀 How to Configure and Run the Code

This project requires absolutely no complex server setups to evaluate. It is designed to run locally right out of the box.

### Prerequisites

- **Python 3.x** installed on your system.
- A standard modern web browser (Chrome, Edge, Safari).

### Step-by-Step Setup Guide

**Step 1: Open the Project Folder**  
Extract the submission folder and open your terminal (Command Prompt, PowerShell, or Mac Terminal) inside the `EcoChampions_Hackathon` directory.

**Step 2: Install Dependencies**  
We use Flask to serve the web application. Install it by running the following command:

```bash
pip install -r requirements.txt

**Step 3: Run the Server
Start the Flask backend application by running:
python app.py

**Step 4: Launch the App
Open your web browser and navigate exactly to:

👉 http://127.0.0.1:5000
```
